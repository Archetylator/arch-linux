# Arch Linux Installation Scripts

Works only in UEFI mode!

Features: 
* LVM on LUKS
* :lock: with dm-crypt
* swap partition
* customize:
  * keyboard layout
  * localization
  * partitions size
  * passwords
  * hostname

The installation script `sh -c "$(curl --location --silent https://goo.gl/ptUyx8)"`

The post-installation script `sh -c "$(curl --location --silent https://goo.gl/PSeJNd)"`
